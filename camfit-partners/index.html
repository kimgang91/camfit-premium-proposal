01 — 현황 진단
해결해야 할 핵심 문제
기존 모델의 구조적 한계를 진단하고, 이를 기반으로 신규 전략을 수립합니다.
1

업주 이탈 — 중복예약

타사 플랫폼과의 중복예약 반복 발생. 업주가 직접 해결해야 하는 구조로 피로 누적 후 탈퇴.

2

MD 영업 동력 저하

캠포너스 폐지 이후 보상 체계 공백. 입점시켜도 방치되는 구조로 영업 의미 상실.

3

성장 정체

프렌들리 신규 DB 유입 감소. 펜션 영역에서 새로운 수익원 확보가 시급한 상황.

08 — 실행 로드맵
3단계 실행 계획
Phase 1 — 즉시 실행 (1~2주)
Quick Win으로 MD 동력 점화
  • 모델 A 수수료 테이블 확정 및 MD 브리핑 자료 작성
  • 슬랙 예약 알림 봇 세팅 (예약 확정 기능 개발 전 임시)
  • MD 대상 프로모션 킥오프 미팅 진행
Phase 2 — 단기 구축 (1~3개월)
시스템으로 차별화
  • 예약 확정 기능 MVP 스펙 정의 및 개발
  • 업주용 월간 리포트 자동 발송 기능 구축
  • MD용 업체별 예약 현황 대시보드 제공
  • S급 펜션 타겟 리스트 10곳 선정 및 공략 시작
Phase 3 — 중기 확장 (3~6개월)
검증 후 확장
  • 모델 A 성과 평가 (업체당 월 예약 5건 이상 여부)
  • 모델 B(MD 프리미엄) 도입 검토
  • 부가서비스 상품 정식 런칭
  • 월 5만원 저가 홈페이지 상품 영업 적용
